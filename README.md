# pro

Personal package manager

pro helps you to kickstart new projects.

Set Github username:

```js
$ pro set user creaturephil
```

Get project directory:

```js
$ pro bootstrap
```

Set repository for files: (must set username first)

```js
$ pro set file files
```

Get file:

```js
$ pro index.html
```

List repositories:

```js
$ pro list user
```

List files:

```js
$ pro list files
```
